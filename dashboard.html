🔧 How to set up Gmail integration

One-time setup (10 minutes):

  1. Go to console.cloud.google.com and create a project
  2. Search "Gmail API" → Enable it
  3. Go to OAuth consent screen → External → add scope gmail.readonly
  4. Go to Credentials → Create OAuth 2.0 Client ID → Web application
  5. Add your domain to Authorized JavaScript origins
  6. Copy the Client ID, open js/config.js and paste it as GOOGLE_CLIENT_ID
  7. Reload the app — the "Connect Gmail" button will work

Your Gmail credentials are never stored. The app only reads email content to find bank transaction alerts.
